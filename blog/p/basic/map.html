map是啥
数学上说，map是映射.
用大白话说，map就是能把一堆中的每件物品与另一堆中的一一对应的魔法.
怎么用
首先，需要一个头文件： 
#include <map>
using namespace std;
然后她的定义是： 
map <A,B> mmp;
其中A和B是类型 (如 int,string,double.vector<int>,vector<vector<int>>...... )
怎么访问呢？
直接当作数组用就行！
也就是说，如果你定义了一个
map <string,int> mmp
就直接 
mmp["OrzDalao"]=233;

进阶用法
类似于set,map也可以： 
 insert();
erase(keyvalue/iterator);
lower_bound(); (->iterator)
upper_bound(); (->iterator)
find(); (->iterator)
clear();
size();
除了这些,map还有一个功能:count(a) - 数数有多少个元素a
只不过...因为map是映射，为集合到集合的一一对应关系，满足互异性，所以......
count只会返回0/1
复杂度
map里面为
 ^$%^%#(!@%^$@*#)
树（好像是一种平衡树，复杂度为 $log_2n$
只不过常数有些大（比我手写的treap慢两倍左右）,但开$O_2$可能会跑得比较快
这东西能干哈？

这样能起到一定的降低代码复杂度的作用。

#define MAXN 12

int n,m;
int a[MAXN][MAXN], b[MAXN][MAXN];

int put[MAXN];
bool allput;
ll Gethash(){
    ll ret = 0;
    allput = 1;
    For(i,m){
        ret *= (ll)n;
        ret += (ll)put[i];
        if( put[i] != n ) allput = 0;
    }
    return ret;
}
map< ll, pii > mem;
pii Dfs( bool player ){
    pii ans;
    if(!player) ans = Mp(-INF,INF);
    else ans = Mp(INF,-INF);

    ll stat = Gethash();
    if(allput) return Mp(0,0);
    if(mem.count(stat)) return mem[stat];

    For(i,m){
        if( put[i] == n  ||  ( put[i] == put[i-1] && i != 1 ) ) continue;

        put[i]++;
        pii tans = Dfs(player^1);
        put[i]--;

        if(!player){
            tans.F += a[put[i]+1][i];
            if( tans.F - tans.S > ans.F - ans.S )
                ans = tans;
        }else{
            tans.S += b[put[i]+1][i];
            if( tans.S - tans.F > ans.S - ans.F )
                ans = tans;
        }
    }

    return mem[stat] = ans;
}

int main(){
    FILEIO();

        For(j,m)
            Read(a[i][j]);
    For(i,n)
        For(j,m)
            Read(b[i][j]);

    pii ans = Dfs(0);
    printf("%d\n",ans.F-ans.S);

    END();
}
拓展：multimap
与map类似，只是一个元素可以存在多次。同理有multiset
此时 count() 的返回值就不一定是0/1了
注意！当使用自定义结构体创建multimap/multiset时一定要写好 != 和 <
否则会出现奇怪错误
